List of allowed configuration values

* Data Type: String, Integer, Float, Boolean, UUID, Timestamp, Date, JSON

* Allowed Column Types: Comma-separated value - undefined, int, float, bool, str, uuid, timestamp, date, json, image. There is no "numeric" type - use "int" and/or "float" instead.

* Tag Hints: Comma-separated value - primary_timestamp, categorical, continuous, prediction, ground_truth, pin_in_deep_dive, possible_segmentation. There is no "label" tag - use "ground_truth" instead.

* Aggregate Arguments Parameter Type: Dataset, Literal, Column
