# Platform Default Metrics
This document provides the list of metrics that are populated out of the box in the Arthur Platform.

## Regression models

* Inference count
* Inference null count
* Categorical Population stability index (PSI) w/ 30 day lookback
* Continuous Population stability index (PSI) w/ 30 day lookback
* Numeric distribution
* Numeric sum
* Category counts
* Mean squared error (MSE)
* Root mean squared error (RMSE)
* Mean absolute error (MAE)

## Binary and multi-class classification models

* Inference count
* Inference null count
* Categorical Population stability index (PSI) w/ 30 day lookback
* Continuous Population stability index (PSI) w/ 30 day lookback
* Numeric distribution
* Numeric sum
* Category counts
* Accuracy, precision, recall, F1 score
* Confusion matrix
* Confusion matrix rates

## GenAI models

* Inference count
* Prompt / response token count
* Guardrail pass & fail rates
* Latency
* Hallucination count
* Hallucination intensity distribution
* Prompt injection count
* Toxicity count
* Toxicity score distribution
* PII count
* Sensitive data count

## Agentic applications

* Inference count
* Trace count
* Tool result
* Tool usage eval
* Tool selection eval
* Query relevance eval
* Response relevance eval
